  • У элемента есть 2 области:
    • клиентская -> content + paddings - (clientWidth, clientHeight)
    • весь box -> content + paddings + border - (offsetWidth, offsetHeight)
    Полоса прокрутки oтъедает часть ширины элемента
  • отступ клиентской области от внешних границ элемента
    (clientLeft, clientTop) (по сути это ширина border)
  • ширина и высота элемента c учетом прокрутки (scrollWidth, scrollHeight)
  • на сколько прокрутили элемент - (scrollTop, scrollLeft)
    В scrollTop и scrollLeft можно присваивать значения
    • координаты offsetTop, offsetLeft отсчитываюся относительно ближайшего родителя у которого свойство position != static или относительно body если такого родителя нет
    • offsetParent содержит cсылку на родительский элемент
    • Размеры элементы автоматически обнуляются если ему присвоить display:none
    • А при visibility: hidden размеры элемента сохраняются
