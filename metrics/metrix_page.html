Ширина/высота видимой области окна.

Метод document.documentElement.clientWidth Метод document.documentElement.clientHeight

Метод window.innerWidth Метод window.innerHeight

Свойства clientWidth / Height возвращают ширина / высота минус ширина полосы прокрутки
window.innerWidth / Height игнорируют наличие полосы прокрутки

Координаты блока относительно области просмотра браузера

Метод getBoundingClientRect() позволяет получить координаты блока.
Значения top и left - относительно верхнего левого угла порта просмотра. !!!

Для получения координат элемента относительно страницы нужно к координатам относительно окна добавить текущую прокрутку. Это свойства window.pageYOffset window.pageXOffset

Как правило пишут такую функцию 

    
        function getCoords(el) {
            const box = el.getBoundingClientRect();
            const top = box.top + (window.pageYOffset || document.documentElement.scrollTop);
            const left = box.left + (window.pageXOffset || document.documentElement.scrollLeft );
            return {
                top: Math.round(top),
                left: Math.round(left)
            }
        }
Выполнить getCoords - расчитать координаты нижнего прямоугольника

Установить прокрутку можно при помощи специальных методов

Например прокрутить страницу на 250px от верхнего края документа document.documentElement.scrollTop = 250

window.scrollTo(pageX,pageY) – абсолютные координаты
Например прокрутить страницу на 100px от верхнего края документа window.srollTo(0, 100)

window.scrollBy(x,y) – прокрутить относительно текущего места.
Например прокрутить страницу на 50px от текущего места вниз
window.scrollBy(0, 50)
